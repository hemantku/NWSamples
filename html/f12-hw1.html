CS631 -- Advanced Programming in the UNIX Environment - Fall 2011 - HW#1

HW#1: code reading

Summary

The objective of this assignment is for you to get used to read (and understand!) code. To this end, please take a look at the NetBSD source code for programs installed under /bin, pick two programs of your liking and inspect the code. Then, read the corresponding code of the same programs in Linux or Open Solaris. Compare the different implementations. What do they have in common? What makes them different?

Deliverables and Due Date

You will submit a single plain text file created with a unix text editor on a unix system called "HW1-<username>" (where "<username>" is your Stevens username). In that file write down (in proper English, spell-checked and with valid grammar) which programs you compared, how you chose them, how you retrieved the source code to the programs in question and what you have found out about the implementations.

The due date for this assignment is 2012-09-10 18:15. Please attach the file to an email sent from your @stevens.edu email address to jschauma@stevens.edu with a subject of "[CS631] HW1".
