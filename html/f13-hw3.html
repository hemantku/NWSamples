CS631 -- Advanced Programming in the UNIX Environment - Fall 2013 - HW#3

HW#3: webserver framework

Summary

The objective of this assignment is for you to lay down the very basic framework for your final project. This ensures that you are not beginning too late in the semester to work on the project and that all of you can make the same progress.

In particular, you are to write the following portions of your final project:

  • the main function, parsing all command-line options and setting all flags
  • the basic code to listen on the given network address and port, accept connections
Your rudimentary server will read one line of text from any connecting client and print out on stdout that line prefixed with the remote client's IP address.

Deliverables and Due Date

You will submit at least three files:

  • main.c - containing the basic functionality
  • net.c - containig any network related functionality
  • Makefile

You will submit a single tar archive named username-hw3.tar that will extract into a directory named after your username.

For example, I would submit the file jschauma-hw3.tar which might extract the files and let me run the program as follows: 

$ tar xvf jschauma-hw3.tar
jschauma
jschauma/main.c
jschauma/net.c
jschauma/Makefile
jschauma/README
$

The due date for this assignment is 2013-11-04 16:00. Please attach the file to an email sent from your @stevens.edu email address to jschauma@stevens.edu with a subject of "[CS631] HW3".
