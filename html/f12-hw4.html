HW#4: yes, even more code reading

Problem Assignment

It is important to be able to read other people's code. Sometimes that code is beautiful, well-written, clear and correct; other times, it is buggy, difficult to understand and convoluted. We don't always have a choice in which code we want to read and have to understand.

To illustrate this, you will each review and grade one of your classmate's midterm submissions. In the directory

http://www.cs.stevens.edu/~jschauma/631/hw4/

you will find a file with your username. Download that file and then evaluate the submission as if you were the instructor. That is, read the code, comment on the good, the bad and the ugly parts; execute the program and verify that all functionality is present and the program works as expected and required.

At the end, determine an appropriate grade for the code in question -- make sure to justify your grade based on your findings.

Deliverables and Due Date
