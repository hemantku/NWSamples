CS631 -- Advanced Programming in the UNIX Environment - Fall 2012 - HW#4

HW#4: sws milestone#2

Problem Assignment

In this assignment, you will build up your HTTP server with the foundation of HTTP support. Using your submission from HW#3 and working with your team, you will begin to stub out the protocol specific components. Naturally, you will split your functionality into appropriate files, at a minimum a file called 'http.c'.

Your team's Milestone#2 (HW#3 was the first milestone) will:

  • correct any shortcomings noted in your HW#3
  • add the file 'http.c', which will include the HTTP protocol specific functionality
  • read input from the client and validate that it follows the HTTP specification
  • return the correct HTTP code for the following use cases:
    • input was a valid HTTP request supported by this server => 200 OK
    • input was a valid HTTP request not supported by this server => 501 Not Implemented
    • input was a valid HTTP request of an HTTP version not supported by this server => 505 Version Not Supported
    • input was an invalid request => 400 Bad Request
  • time out open client connections after a suitable amount of time, yielding a 522 Connection Timed Out status

Deliverables and Due Date

Your team will submit one tar archive named HW4-$team.tar (where $team is the name of your team). In addition to the mandatory Makefile and the source files, you will also include a file GitLog, which contains the full output of the command 'git log'.
